Autonomous Terminal Trucks (ATT)

Led the development of autonomous driving systems for port terminal operations, delivering production-grade solutions across multiple international deployments.

Architected and implemented real-time perception systems for autonomous terminal trucks operating in busy port environments. The system handles container transport operations 24/7 with minimal human intervention.

Port terminals present unique challenges: dynamic environments with heavy machinery, varying weather conditions, 24/7 operations, and strict safety requirements. The perception system needed to achieve near-perfect accuracy while maintaining real-time performance.

Developed a multi-modal sensor fusion system combining LiDAR and camera data for robust perception. Implemented deep learning-based object detection with custom-trained models optimized for port environments and real-time inference.

Key Achievements

  • 99.9% detection accuracy
  • Sub-100ms latency
  • 1000+ containers daily
  • Multi-sensor fusion
  • All-weather operation
  • 24/7 autonomous operation
