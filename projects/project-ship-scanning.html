Ship Profile Scanning System (SPSS)

Developed a LiDAR and camera-based system for real-time vessel profiling, detecting container stacks and vessel structures to automate crane operation planning.

Architected 3D point cloud processing pipeline for vessel scanning, enabling automated identification of container positions on vessels for crane operation planning.

Vessel scanning requires processing massive 3D point clouds in real-time while vessels are docked. Maritime conditions including salt spray, varying lighting, and vessel movement add complexity to accurate profiling.

Developed container stack detection and classification algorithms with multi-sensor fusion for comprehensive vessel profiling. Integrated real-time updating during operations for continuous accuracy.

Key Achievements

  • 99.5% detection accuracy
  • 30-second scan processing
  • 60% planning time reduction
  • Real-time profile updates
  • Maritime conditions robust
  • Crane system integration
