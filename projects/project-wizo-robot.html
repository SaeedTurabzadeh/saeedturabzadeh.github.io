Humanoid Robot (WIZO)

The world's first 3D printed humanoid robot. A multi-functional robot featuring embedded vision for real-time object and facial recognition, integrated with NLP for user interaction.

Developed computer vision algorithms for real-time object detection and facial recognition. Integrated NLP capabilities for natural human-robot interaction on embedded hardware.

Creating an affordable humanoid robot with advanced perception capabilities required optimizing deep learning algorithms for embedded hardware with limited computational resources while maintaining real-time performance.

Implemented efficient vision algorithms optimized for embedded constraints. Combined 3D printing for accessible manufacturing with advanced AI for perception and interaction capabilities.

Key Achievements

  • 95%+ facial recognition
  • Real-time object tracking
  • Voice command NLP
  • 3D printed construction
  • International recognition
  • Published research
